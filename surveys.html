Evaluation Method

Surveys

Gather structured feedback from many users quickly. Great for validating hypotheses and identifying patterns at scale.

Time Required

2-4 Weeks

Participants Needed

100+ Users

Difficulty Level

Intermediate

Overview

Surveys collect structured feedback from large user groups through standardised questions. This method provides both quantitative and qualitative data that can validate hypotheses, reveal patterns, and identify trends across your user base.

Usage

When to use

  • Gathering feedback at scale
  • Validating hypothesis
  • Measuring satisfaction
  • Understanding preferences
  • Identifying trends
  • Collecting demographics
  • Benchmarking progress
  • Prioritising features

When not to use

  • For deep user understanding
  • Testing usability
  • Exploring new concepts
  • When sample size is small
  • For complex workflows
  • Understanding behaviour
  • Gathering emotional responses

Analysis

Example Questions

  • "How satisfied are you with this feature?"
  • "How likely are you to recommend our product?"
  • "Which of these features do you use most?"
  • "What would improve your experience?"
  • "How often do you encounter this issue?"
