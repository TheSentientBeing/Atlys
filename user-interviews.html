Discovery Method

User Interviews

One-on-one conversations that reveal deep insights about user needs, behaviors, and pain points.

Time Required

2-3 weeks

Participants Needed

8-12 users

Difficulty Level

Intermediate

Overview

User interviews are structured conversations designed to uncover deep insights into users' behaviours, needs, motivations, and pain points. This method allows for detailed exploration of user experiences and provides rich, qualitative data through direct dialogue.

Usage

When to use

  • Understanding user needs
  • Exploring behaviours
  • Uncovering motivations
  • Investigating pain points
  • Gathering stories
  • Validating assumptions
  • Building empathy
  • Exploring new areas

When not to use

  • Testing usability
  • Gathering metrics
  • Validating designs
  • When time is limited
  • For large sample sizes
  • Testing functionality
  • When bias risk is high

Process

1

Interview Planning

  • Define research goals
  • Create discussion guide
  • Identify user segments
  • Choose interview format
  • Plan session structure
  • Prepare materials
2

Participant Recruitment

  • Define screening criteria
  • Source participants
  • Screen candidates
  • Schedule sessions
  • Send confirmations
  • Arrange incentives
3

Interview Preparation

  • Review discussion guide
  • Set up recording
  • Prepare environment
  • Test equipment
  • Review participant details
  • Brief stakeholders
4

Conducting Interviews

  • Build rapport
  • Ask open questions
  • Listen actively
  • Probe responses
  • Manage Time
  • Take Notes
  • Avoid leading questions
5

Documentation

  • Record key quotes
  • Note observations
  • Capture context
  • Document non-verbals
  • Summarise findings
  • Identify themes
6

Analysis

  • Record key quotes
  • Note observations
  • Capture context
  • Document non-verbals
  • Summarise findings
  • Identify themes
7

Analysis & Synthesis

  • Review recordings
  • Code responses
  • Identify patterns
  • Map insights
  • Create frameworks
  • Draw conclusions

Example Questions

  • "Tell me about a time when..."
  • "How do you currently..."
  • "What challenges do you face..."
  • "Walk me through your process..."
  • "Why is that important to you?"
