Discovery Method

Eye Tracking

Use specialized equipment to record users' eye movements and fixation points. Reveals unconscious viewing patterns and heat maps.

Time Required

2-3 Weeks

Participants Needed

20-30 Users

Difficulty Level

Advanced

Overview

Eye tracking studies use specialized hardware and software to record where users look, for how long, and in what order. This method provides objective data about visual attention, revealing unconscious behaviours and helping optimize interface design for natural viewing patterns.

Usage

When to use

  • Evaluating visual hierarchies
  • Testing advertisement effectiveness
  • Optimising content placement
  • Validating navigation paths
  • Assessing search patterns
  • Measuring cognitive load
  • Comparing layout alternatives

When not to use

  • When equipment access is limited
  • For testing non-visual interactions
  • When gathering emotional feedback
  • For mobile/field testing
  • With users who have visual impairments
  • When qualitative insights are primary goal

Process

1

Study Design

  • Define research objectives
  • Select testing scenarios
  • Create task flows
  • Design stimulus materials
  • Plan calibration process
  • Set success metrics
2

Equipment Setup

  • Calibrate eye tracking hardware
  • Configure test environment
  • Test recording settings
  • Set eye tracking environment
  • Plan for contingencies
  • Prepare backup systems
3

Participant Preparation

  • Schedule participants
  • Review test plan
  • Brief participants
  • Set expectations
  • Run calibration tests
  • Address concerns
4

Data Collection

  • Record eye movements
  • Track fixation points
  • Monitor participants
  • Note observations
  • Document special cases
  • Save data securely
5

Initial Analysis

  • Generate heat maps
  • Create scan paths
  • Calculate fixation times
  • Map viewing patterns
  • Note initial findings
6

Advanced Analysis

  • Compare user segments
  • Analyze gaze plots
  • Evaluate attention spans
  • Study viewing anomalies
  • Identify common patterns
7

Reporting & Visualization

  • Create visual reports
  • Generate heat maps
  • Plot gaze patterns
  • Document key findings
  • Present recommendations

Example Questions

  • "Where do users look first on the page?"
  • "How long do users focus on key elements?"
  • "What is the typical scanning pattern?"
  • "How does visual attention compare with task success?"

Pairs well with

Usability Testing

Why?

Combine eye tracking data with traditional usability metrics to understand not only what users look at and see, but why they look and see that.

A/B Testing

Why?

Use eye tracking insights to learn if design hypotheses actually improve the perception of target content.
