Discovery Method

User Personas

Synthesize research findings into archetypal users that represent different user segments, their goals, behaviors, and pain points. Essential for building user empathy.

Time Required

2-4 weeks

Participants Needed

Based on prior research

Difficulty Level

Advanced

Overview

User personas are research-based representations of your key user groups, combining quantitative and qualitative data findings. These deliverables encapsulate real research findings into models that combine user shared behaviors, traits, pain points, and common contexts to help teams stay focused throughout the design process.

Usage

When to use

  • Guiding design decisions
  • Building team empathy
  • Prioritizing features
  • Defining target users
  • Aligning stakeholders
  • Creating scenarios
  • Informing strategy
  • Focusing development

When not to use

  • Without user research
  • For technical specifications
  • When segments unclear
  • Testing usability
  • Making design choices
  • For edge cases only
  • When data is limited

Process

1

Research Review

  • Gather existing research
  • Review user data
  • Collect analytics
  • Analyze interviews
  • Examine surveys
  • Compile observations
2

Pattern Identification

  • Identify behaviors
  • Map motivations
  • Group demographics
  • Note common goals
  • List pain points
  • Find correlations
3

Segment Definition

  • Define key segments
  • Validate groupings
  • Create coverage
  • Identify overlaps
  • Prioritise segments
  • Document criteria
4

Persona Development

  • Create narratives
  • Write backgrounds
  • Define goals
  • List frustrations
  • Add quotes
  • Select images
  • Build scenarios
5

Validation

  • Review with stakeholders
  • Check against research
  • Validate with users
  • Test completeness
  • Verify accuracy
  • Refine details
6

Implementation

  • Share with team
  • Create resources
  • Plan usage
  • Track adoption
  • Monitor effectiveness
  • Update as needed

Example Questions

  • "What patterns emerge from our research?"
  • "Which user groups are most important?"
  • "What motivates this person?"
  • "How does this persona make decisions?"
  • "What are their key pain points?"
